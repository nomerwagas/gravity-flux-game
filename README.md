Technical Requirements
1. Project Setup
Create your project using:

npx create-expo-app one-button-game
Run using:

npx expo start
-------------------------------------------------------------------------
2. Required Features
Your game must include:

Start Screen

Game Screen

Game Over Screen

Score counter

Restart button (after Game Over)

Basic UI styling (not default unstyled components)
-------------------------------------------------------------------------

Required React Native Concepts
You must demonstrate use of:

useState

useEffect

Conditional rendering

A Touchable component (Pressable, TouchableOpacity, etc.)

At least one timed or animated mechanic:

setInterval

setTimeout

Animated

or an Expo module such as audio or haptics

-------------------------------------------------------------------------

Example Game Concepts (Do Not Copy Exactly)
Tap at the correct time to stop a moving indicator

Tap to jump over obstacles (miss once = death)

Tap to charge power; overcharge results in failure

Tap to maintain balance within a range

Tap to dodge randomly appearing hazards

You are encouraged to create an original idea.
